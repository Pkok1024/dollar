File Upload API Documentation

The File Upload API allows you to upload files to the server, and it returns the URL of the uploaded file.

Base URL

https://your-server-base-url.com/api/upload-to-discord

Endpoint

Upload File

  • Method: POST
  • Endpoint: /api/upload-to-discord

Request

Method

POST

Content-Type

multipart/form-data

Parameters

| Parameter  | Type  | Description                        |
|------------|-------|------------------------------------|
| file     | File  | The file to be uploaded.           |

Example

Using Axios for a POST request:

const formData = new FormData();
formData.append('file', file); // Assuming you have a file input element

axios.post('https://your-server-base-url.com/api/upload-to-discord', formData)
  .then(response => {
    console.log('Response:', response.data);
  })
  .catch(error => {
    console.error('Error:', error);
  });
